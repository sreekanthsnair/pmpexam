PMP Exam Focus 2025: Pass Essentials

Targets 2025 PMP exam (180 questions, 175 scored). Focus: People (42%, ~75 Qs), Process (50%, ~90 Qs), Business Environment (8%, ~15 Qs). 50% agile/hybrid. Key topics only.

1. People (42% - Team, Stakeholders, Leadership)

Must-Know: Conflict, team performance, stakeholder mgmt - situational Qs dominate.

  • Conflict Resolution:
    • Collaborate: Best, win-win, team solves together.
    • Compromise: Quick fix, both give up something.
    • Avoid: Delay non-critical issues.
    • Force: Last resort, PM decides.
    Exam Tip: Pick collaboration unless time-critical.
  • Team Performance:
    • First Step: Discuss issues, coach (not fire/escalate).
    • Motivation: Recognize effort (not just pay).
    • Agile: Self-organizing, no micromanaging.
    Exam Tip: Support > punish.
  • Stakeholder Engagement:
    • Identify: Use register, power/interest grid.
    • Engage: Tailor comm (don’t ignore/escalate first).
    • Change Requests: Explain process, don’t bypass.
    Exam Tip: Qs test proactive comm.
  • Tools: RACI (Responsible, Accountable, Consulted, Informed), comm plan.

Key Q Example: Team argues over tasks - Facilitate discussion, not escalate.

2. Process (50% - Execution, Metrics, Risks)

Must-Know: EVM, critical path, risk, agile, change - heaviest exam weight.

  • EVM (Earned Value Management):
    • CV = EV - AC: + = under budget, - = over.
    • SV = EV - PV: + = ahead, - = behind.
    • CPI = EV / AC: >1 = good cost, <1 = overrun.
    • SPI = EV / PV: >1 = ahead, <1 = behind.
    • EAC = BAC / CPI: Forecast if trend continues.
    • TCPI = (BAC - EV) / (BAC - AC): Efficiency to finish on budget.
    Exam Tip: 5-10 Qs, memorize formulas. Ex: EV = $200K, AC = $250K, CPI = 0.8.
  • Critical Path:
    • Longest path, zero float - delays here delay project.
    • Crash: Add resources, ↑ cost. Fast-track: Overlap, ↑ risk.
    Exam Tip: Qs give network diagram, calculate path (e.g., A-B-D = 12d).
  • Risk Management:
    • Steps: Identify (register), Analyze (P × I = EMV), Respond (avoid, mitigate, transfer, accept).
    • Reserves: Contingency (known unknowns), Management (unknown unknowns).
    • Risk occurs: Use planned response first.
    Exam Tip: EMV Qs (e.g., 30% × $50K = $15K).
  • Agile/Hybrid:
    • Sprint: 1-4 weeks, goal fixed, changes to backlog.
    • Roles: Product Owner (backlog), Scrum Master (process), Team (delivers).
    • Events: Planning, Daily Scrum (15 min), Review, Retrospective.
    • Hybrid: Predictive (scope fixed) + Agile (iterative).
    Exam Tip: 50% of Qs, defer mid-sprint changes.
  • Change Control:
    • Process: Request → CCB → Update plans (schedule, scope, etc.).
    • No shortcuts, even for sponsors.
    Exam Tip: Always pick formal process.
  • Quality:
    • QA: Improve process. QC: Inspect deliverables.
    • Tools: Pareto (80/20), Ishikawa (causes), Control Chart (trends).
    Exam Tip: Match tool to purpose.
  • Procurement:
    • Fixed-price: Clear scope, seller risk.
    • Cost-plus: Flexible, buyer risk.
    • T&M: Moderate risk, time-based.
    Exam Tip: Qs test risk ownership.

Key Q Example: Risk occurs - Implement response, not notify first.

3. Business Environment (8% - Strategy, Value)

Must-Know: Alignment, compliance - fewer Qs but critical.

  • Strategic Alignment:
    • Ensure project fits org goals.
    • Shift happens: Assess impact, submit change (not terminate first).
    Exam Tip: Qs test proactive adjustment.
  • Benefits:
    • Define value (e.g., ROI), track post-project.
    • Business case justifies project.
  • Compliance:
    • Legal/ethical rules (PMI Code: fairness, honesty).
    • Qs test ethics (e.g., no bribes).
    Exam Tip: Always choose ethical option.

Key Q Example: Market shifts - Assess and change, not continue.

4. Exam Must-Knows

Critical for Passing: Focus here to nail 2025 PMP.

  • Formulas: EVM (CV, SV, CPI, SPI, EAC, TCPI), comm channels: n(n-1)/2.
  • Agile: 50% of exam - sprint rules, roles, defer changes.
  • Risk: EMV, responses, reserves - situational Qs.
  • Change: Always formal process, no shortcuts.
  • Tools: Pareto (prioritize), Ishikawa (causes), Gantt (schedule).
  • Ethics: PMI Code - fairness, honesty, responsibility.
  • Qs Breakdown: Situational (60-70%), formulas (5-10%), tools (10-15%), ethics (5-10%).
  • Tip: Eliminate distractors - pick PMI’s process over gut.

Study Plan: 1 hr/day - 20 min formulas, 20 min agile, 20 min risk/change.
